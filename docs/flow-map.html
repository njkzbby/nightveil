Forward Secrecy

Каждый запрос генерирует новую эфемерную пару ephPriv/ephPub. Даже если userPriv скомпрометирован, прошлые сессии не дешифруются.

Anti-Replay

Timestamp в токене с окном +-60s. Повторная отправка перехваченного токена за пределами окна отклоняется.

User Binding

Double ECDH: sharedEph доказывает владение эфемерным ключом, sharedUser доказывает принадлежность к зарегистрированному пользователю.

Upload (POST)

Данные нарезаются на чанки по maxChunkSize (8-14KB). Каждый чанк — отдельный POST с seq=N. Сервер переупорядочивает по sequence number.

Download (GET)

Один долгоживущий GET-запрос. Сервер стримит данные в response body с Transfer-Encoding: chunked и постоянным flush.

Anti-DPI

DPI видит обычные HTTP GET/POST на CDN-подобные пути. Чанки <15KB — под порогом обнаружения ТСПУ. Пути рандомизируются каждые 30 минут.

7 Throttle Detection State Machine

Конечный автомат детектора троттлинга: метрики, пороги, переходы состояний, реакции.

stateDiagram-v2
    [*] --> Normal : Startup (baseline learning)

    Normal --> Suspected : RTT > 500ms AND > 3x baseline\nOR throughput drop > 70%
    Suspected --> Normal : Metrics cleared\n(suspectedCount = 0)
    Suspected --> Throttled : suspectedCount >= 3\n(3 consecutive suspected samples)
    Throttled --> Normal : Metrics cleared\n(suspectedCount = 0)

    state Normal {
        [*] --> Monitoring
        Monitoring --> Monitoring : RecordRTT() / RecordThroughput()\nupdates sliding window (20 samples)
        Monitoring --> BaselineLearning : First 5 samples
        BaselineLearning --> Monitoring : Baseline established\nmedianRTT + avgThroughput
    }

    state Suspected {
        [*] --> Counting
        Counting --> Counting : Each bad sample:\nsuspectedCount++
    }

    state Throttled {
        [*] --> Active
        Active --> ScaleUp : Adaptive: conns *= 2 (max 8)
        ScaleUp --> Rotate : Emergency rotation triggered
        Rotate --> Active : New params applied
    }

Normal State

Collecting baseline from first 5 samples. medianRTT and avgThroughput established. Window: last 20 samples.

Suspected State

RTT spike or throughput drop detected. suspectedCount increments. Need 3 consecutive hits to confirm. Single good sample resets to Normal.

Throttled State

Confirmed. Adaptive doubles parallel conns: 1 -> 2 -> 4 -> 8. Emergency rotation fires: all params + paths randomized immediately.

MetricThresholdCondition
RTT Spike (absolute)500msRecent RTT > RTTSpikeMs
RTT Spike (relative)3x baselineRecent RTT > 3 * medianRTT
Throughput Drop70%Drop % = (1 - recent/baseline) * 100
Confirmation3 samplesConsecutive suspected evaluations
Window Size20 samplesSliding window for RTT and throughput
Scale Factor2x per step1 -> 2 -> 4 -> 8 parallel conns

Padding

Wire format: [2B realLen][2B padLen][data][random]. Скрывает реальный размер пакетов. Max frame: 65535 bytes.

Jitter

Рандомные задержки 5-100ms на каждый Write. Ломает cross-layer RTT fingerprinting (NDSS 2025 атака). Без jitter: TCP RTT 20ms, TLS 120ms — дельта выдаёт прокси.

Traffic Shaping

Profiles: Browsing (bursts + pauses), Streaming (steady 500KB/s), Idle (1-5s gaps). Auto-detect по read/write ratio.

Cover Traffic

Фоновые GET/HEAD запросы каждые 2-10s к CDN/portal endpoints. 70% GET, 30% HEAD. Маскирует паттерн реального трафика.
